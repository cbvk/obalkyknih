ObalkyKnih:

LibraryThing: podle LibraryThing JSON Books API dokumentace neni mozne v databazi vyhledavat, lze ziskavat detaily (nazev, autor, obalka) pouze o knihach, ktere ma uzivatel na svem konte.

GoogleBooks: lze vyhledavat podle ISBN (a taky OCLC nebo LCCN), lze nalinkovat jejich thumbnail obrazek, lze odkazovat na Google Books stranku o knizce. Nelze k ISBN zjistit bibliograficke udaje. Slo by teoreticky stahnout a parsovat z odkazovane stranky (nazev v <title> tagu, autor s "By " prefixem) ale ...

Amazon: nejaky slozity postup (registrace dvou uctu,... ) ale melo by jit dle ISBN (ASIN) ziskat nazev (item.getItemAttributes().getTitle()) pres Java API. Existuji i nejake perl moduly (DBD::Amazon, Net::Amazon::AWIS, Amazon: http://www.hccp.org/amazon-asin-isbn.html http://www.mularien.com/blog/2008/04/10/5-minute-guide-to-the-java-amazon-associates-web-service-api/

ISBNdb.com 3.5M knih, 1M autoru, perl modul WebService::ISBNDB pro pouziti. Popis.
